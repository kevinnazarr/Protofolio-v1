# Protofolio v1

Portofolio berbasis web pertama saya yang dibangun menggunakan HTML, CSS, dan JavaScript.
Project ini dibuat untuk menampilkan profil pribadi, daftar project, dan skill dalam pengalaman satu halaman yang sederhana, interaktif, dan responsif.

## Tentang Project

Protofolio v1 adalah website portofolio personal yang dirancang untuk memperkenalkan identitas saya secara lebih profesional di web.
Dengan tampilan yang modern dan navigasi berbasis section, website ini memudahkan pengunjung untuk melihat informasi utama tanpa berpindah halaman.

## Fitur Utama

- Tampilan single-page portfolio
- Navigasi section yang mudah digunakan
- Efek visual interaktif seperti background animasi dan particle
- Menu mobile yang responsif
- Form kontak dengan feedback sederhana

## Teknologi

- HTML
- CSS
- JavaScript
